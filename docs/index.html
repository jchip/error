@jchip/error

errors

Utilities for node.js errors

API reference: https://jchip.github.io/error

node.js 15+ has built-in AggregateError

Examples

cleanErrorStack

 
import { cleanErrorStack } from "@jchip/error";

try {
  require("oops");
} catch (err) {
  console.log(cleanErrorStack(err));
}

aggregateErrorStack
